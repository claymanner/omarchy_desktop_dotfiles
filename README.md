# omarchy_desktop_dotfiles
This is the dotfiles for my Omarchy Desktop

# Bashrc
- Changes here are for launching tmux on start

# Hyprenv
- This changes the default screenshot file to be the nextcloud screenshot folder

# Ghostty
- Just changing the default font size

# nvim
- Changing the appearance for relative lines, and fixing autocomplete with blink so that enter does not complete, you instead use C-y

# tmux
- Changes the leader to C-f, and changes appearance, as well as some other keybinds

# waybar
- Changes the clock to show the date on the main clock

# xcompose
- Changes the name and email on caps - space - e and n
- Add my work email to caps - space - w
